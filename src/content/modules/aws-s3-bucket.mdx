Provision an S3 bucket with best practices:

- Private ACL, block public access
- Server-side encryption (SSE-S3)
- Versioning and lifecycle policies (optional)

## Usage

### Minimal

```hcl
module "bucket" {
  source = "./modules/aws-s3-bucket"
  name   = "my-app-artifacts"
}
```

### Production

```hcl
module "bucket" {
  source = "./modules/aws-s3-bucket"
  name   = "my-app-artifacts"

  versioning = true

  lifecycle_rules = [{
    id      = "cleanup-old-versions"
    enabled = true
    noncurrent_version_expiration_days = 30
  }]
}
```
